# Tetris Web app
Coding challenge for Affinity

## Notes
This is an Angular application created from angular-seed boilerplate.

## How to use
### Build
```
npm start
```
The game will run at localhost:8000

## Controls
P: Play/Pause

LeftArrow: Left

RightArrow: Right

UpArrow: Rotate

DownArrow: Down

Space: Hard Drop
